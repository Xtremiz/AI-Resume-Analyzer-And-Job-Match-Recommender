# Resume Classification Dataset

This dataset contains resume data for text classification and NLP-based career analysis.  
Each record represents a resume with details like job category, resume text, extracted skills, and years of experience.

## Dataset Information

- **resume_id**: Unique identifier for each resume
- **category**: Job category or field (e.g., HR, Data Science, IT, Sales, etc.)
- **resume_text**: The textual content of the resume
- **skills_list**: List of extracted key skills
- **experience_years**: Total experience (in years)

## Use Cases

- Resume classification using NLP
- Skills extraction and keyword analysis
- Machine learning projects for HR Tech and recruitment
- Building recommendation systems for job matching

## File Information

- `resume.csv` — Contains all the data
- Total columns: 5
